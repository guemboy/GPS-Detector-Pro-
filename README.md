วิธีการใช้งานในสถานการณ์จริง (เพื่อหา GPS Tracker):
ถือเครื่องให้นิ่งที่สุด: ในขณะที่เลื่อนมือถือไปตามจุดต่างๆ ของรถ ให้เลื่อนช้าๆ เหมือนฉีดสเปรย์

สังเกต "Peak": ถ้าเลื่อนผ่านจุดหนึ่งแล้วตัวเลขดีดจาก 0 ไป 40-50 แล้วพอกลับมาที่เดิมก็ดีดอีก แสดงว่า "ตรงนั้นมีแม่เหล็กแน่นอน"

ระวังจุดหลอก: ลำโพงรถยนต์ (บริเวณประตู) และ ไดชาร์จ (ในห้องเครื่อง) จะมีแม่เหล็กแรงสูงอยู่แล้ว ซึ่งแอปจะดังเตือนเป็นปกติครับ
